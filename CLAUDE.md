# JCD Royal Treats - Pastry E-commerce Website

## About the Project
A small business website for selling pastries including banana bread, 
puff puff, chin chin, peanut burger, meat pie, fish roll, and more.

## Tech Stack
- Frontend: HTML, CSS, JavaScript (or React)
- Backend: ASP.NET Core / Node.js
- Database: SQL Server / Supabase
- Payments: Stripe or Paystack
- Auth: User login and registration

## Business Rules
- Website must look warm, inviting and appetizing
- Always make pages mobile responsive
- Products must have images, descriptions and prices
- Customers should be able to add items to a cart
- Include an order/checkout system
- Admin should be able to add/edit/delete products

## Code Rules
- Write clean, production-ready code
- Always handle errors gracefully
- Keep code well commented
- Follow MVC pattern
